Bem Vindo à página !


Olá, essa é uma página teste feita para exemplificar o uso da linguagem HTML.

Dentro desta página há o código "!DOCTYPE", que informa o tipo de documento utilizado para criar a página, logo, nesse caso,

o código é seguido do nome HTML resultando em "!DOCTYPE html" entre os sinais de menor que e maior que (< >).

Já os elementos "html", "head", "title" e " "body", guardam:

  • html:Todo o conteúdo da página html
  • head: Os metadados da página web, estas não são visíveis para o usuário, pois é a forma em que a página
  • title: O título do documento HTML.
  • body: Todo o corpo do código, como as tags, por exemplo.

Enquanto o "h1" e "p" são respectivamente usados para criar títulos e parágrafos dentro do site.
